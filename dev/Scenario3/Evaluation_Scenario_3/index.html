Question 1

Setup Model

  1. Begin with a basic SIR model without vital dynamics. Calibrate the model parameters using data on cases during the ‘training period’. Then evaluate the model during the out-of-sample ‘test period’.

To get started with the code, we implemented the basic SIR without vital dynamics directly in ModelingToolkit.jl. This is a version that was written by an epidemiologist at Microsoft Pandemic, Simon Frost, who has become a fan of the TA3 automated simulation tools and wrote an entire repository of tutorials for this software. It is found at https://github.com/epirecipes/sir-julia.

In there is an SIR without vital dynamics which we took in full.

@parameters t β=0.05 c=10.0 γ=0.25
@variables S(t)=990.0 I(t)=10.0 R(t)=0.0
∂ = Differential(t)
N = S + I + R # This is recognized as a derived variable
eqs = [∂(S) ~ -β * c * I / N * S,
    ∂(I) ~ β * c * I / N * S - γ * I,
    ∂(R) ~ γ * I];

@named sys = ODESystem(eqs);

\[ \begin{align} \frac{\mathrm{d} S\left( t \right)}{\mathrm{d}t} =& \frac{ - c \beta I\left( t \right) S\left( t \right)}{I\left( t \right) + R\left( t \right) + S\left( t \right)} \\ \frac{\mathrm{d} I\left( t \right)}{\mathrm{d}t} =& \frac{c \beta I\left( t \right) S\left( t \right)}{I\left( t \right) + R\left( t \right) + S\left( t \right)} - \gamma I\left( t \right) \\ \frac{\mathrm{d} R\left( t \right)}{\mathrm{d}t} =& \gamma I\left( t \right) \end{align} \]

tspan = (0.0, 40.0)
prob = ODEProblem(sys, [], tspan);
sol = solve(prob);

Question 2: Add Hospitalizations and Deaths

Question 2 involves doing the same analysis as question one but on the SIR model with hopsitalizations and deaths included. To establish unit tests, we first showcase building the model and solving inverse problems using the ModelingToolkit version of the model.

@parameters t β=0.1 c=10.0 γ=0.25 ρ=0.1 h=0.1 d=0.1 r=0.1
@variables S(t)=990.0 I(t)=10.0 R(t)=0.0 H(t)=0.0 D(t)=0.0
∂ = Differential(t)
N = S + I + R + H + D # This is recognized as a derived variable
eqs = [∂(S) ~ -β * c * I / N * S,
    ∂(I) ~ β * c * I / N * S - γ * I - h * I - ρ * I,
    ∂(R) ~ γ * I + r * H,
    ∂(H) ~ h * I - r * H - d * H,
    ∂(D) ~ ρ * I + d * H];

@named sys2 = ODESystem(eqs);

\[ \begin{align} \frac{\mathrm{d} S\left( t \right)}{\mathrm{d}t} =& \frac{ - c \beta I\left( t \right) S\left( t \right)}{D\left( t \right) + H\left( t \right) + I\left( t \right) + R\left( t \right) + S\left( t \right)} \\ \frac{\mathrm{d} I\left( t \right)}{\mathrm{d}t} =& \frac{c \beta I\left( t \right) S\left( t \right)}{D\left( t \right) + H\left( t \right) + I\left( t \right) + R\left( t \right) + S\left( t \right)} - h I\left( t \right) - \gamma I\left( t \right) - \rho I\left( t \right) \\ \frac{\mathrm{d} R\left( t \right)}{\mathrm{d}t} =& r H\left( t \right) + \gamma I\left( t \right) \\ \frac{\mathrm{d} H\left( t \right)}{\mathrm{d}t} =& h I\left( t \right) - d H\left( t \right) - r H\left( t \right) \\ \frac{\mathrm{d} D\left( t \right)}{\mathrm{d}t} =& d H\left( t \right) + \rho I\left( t \right) \end{align} \]

Question 3: Add Vaccinations

Question 3 is the same analysis as questions 1 and 2 done on a model with vaccination added. In order to build unit tests for the analysis and functionality, we started by building the model with vaccine by hand, awaiting a swap to the version from TA2.

@parameters t β=0.1 c=10.0 γ=0.25 ρ=0.1 h=0.1 d=0.1 r=0.1 v=0.1
@parameters t β2=0.1 c2=10.0 ρ2=0.1 h2=0.1 d2=0.1 r2=0.1
@variables S(t)=990.0 I(t)=10.0 R(t)=0.0 H(t)=0.0 D(t)=0.0
@variables Sv(t)=990.0 Iv(t)=10.0 Rv(t)=0.0 Hv(t)=0.0 Dv(t)=0.0
@variables I_total(t)

∂ = Differential(t)
N = S + I + R + H + D + Sv + Iv + Iv + Hv + Dv # This is recognized as a derived variable
eqs = [∂(S) ~ -β * c * I_total / N * S - v * Sv,
    ∂(I) ~ β * c * I_total / N * S - γ * I - h * I - ρ * I,
    ∂(R) ~ γ * I + r * H,
    ∂(H) ~ h * I - r * H - d * H,
    ∂(D) ~ ρ * I + d * H, ∂(Sv) ~ -β2 * c2 * I_total / N * Sv + v * Sv,
    ∂(Iv) ~ β2 * c2 * I_total / N * Sv - γ * I - h2 * I - ρ2 * I,
    ∂(Rv) ~ γ * I + r2 * H,
    ∂(Hv) ~ h2 * I - r2 * H - d2 * H,
    ∂(Dv) ~ ρ2 * I + d2 * H, I_total ~ I + Iv,
];

@named sys3 = ODESystem(eqs)
sys3 = structural_simplify(sys3)

\[ \begin{align} \frac{\mathrm{d} S\left( t \right)}{\mathrm{d}t} =& \frac{ - c \beta I_{total}\left( t \right) S\left( t \right)}{2 \mathrm{Iv}\left( t \right) + D\left( t \right) + \mathrm{Dv}\left( t \right) + H\left( t \right) + \mathrm{Hv}\left( t \right) + I\left( t \right) + R\left( t \right) + S\left( t \right) + \mathrm{Sv}\left( t \right)} - v \mathrm{Sv}\left( t \right) \\ \frac{\mathrm{d} I\left( t \right)}{\mathrm{d}t} =& \frac{c \beta I_{total}\left( t \right) S\left( t \right)}{2 \mathrm{Iv}\left( t \right) + D\left( t \right) + \mathrm{Dv}\left( t \right) + H\left( t \right) + \mathrm{Hv}\left( t \right) + I\left( t \right) + R\left( t \right) + S\left( t \right) + \mathrm{Sv}\left( t \right)} - h I\left( t \right) - \gamma I\left( t \right) - \rho I\left( t \right) \\ \frac{\mathrm{d} R\left( t \right)}{\mathrm{d}t} =& r H\left( t \right) + \gamma I\left( t \right) \\ \frac{\mathrm{d} H\left( t \right)}{\mathrm{d}t} =& h I\left( t \right) - d H\left( t \right) - r H\left( t \right) \\ \frac{\mathrm{d} D\left( t \right)}{\mathrm{d}t} =& d H\left( t \right) + \rho I\left( t \right) \\ \frac{\mathrm{d} \mathrm{Sv}\left( t \right)}{\mathrm{d}t} =& v \mathrm{Sv}\left( t \right) + \frac{ - c2 {\beta}2 I_{total}\left( t \right) \mathrm{Sv}\left( t \right)}{2 \mathrm{Iv}\left( t \right) + D\left( t \right) + \mathrm{Dv}\left( t \right) + H\left( t \right) + \mathrm{Hv}\left( t \right) + I\left( t \right) + R\left( t \right) + S\left( t \right) + \mathrm{Sv}\left( t \right)} \\ \frac{\mathrm{d} \mathrm{Iv}\left( t \right)}{\mathrm{d}t} =& \frac{c2 {\beta}2 I_{total}\left( t \right) \mathrm{Sv}\left( t \right)}{2 \mathrm{Iv}\left( t \right) + D\left( t \right) + \mathrm{Dv}\left( t \right) + H\left( t \right) + \mathrm{Hv}\left( t \right) + I\left( t \right) + R\left( t \right) + S\left( t \right) + \mathrm{Sv}\left( t \right)} - h2 I\left( t \right) - \gamma I\left( t \right) - {\rho}2 I\left( t \right) \\ \frac{\mathrm{d} \mathrm{Rv}\left( t \right)}{\mathrm{d}t} =& r2 H\left( t \right) + \gamma I\left( t \right) \\ \frac{\mathrm{d} \mathrm{Hv}\left( t \right)}{\mathrm{d}t} =& h2 I\left( t \right) - d2 H\left( t \right) - r2 H\left( t \right) \\ \frac{\mathrm{d} \mathrm{Dv}\left( t \right)}{\mathrm{d}t} =& d2 H\left( t \right) + {\rho}2 I\left( t \right) \end{align} \]
